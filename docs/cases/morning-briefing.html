날씨, 시장 동향, 뉴스 요약이 매일 아침 텔레그램으로 도착합니다. 설치 3분, 코드 0줄.

내장 스킬
시나리오

출근길, 폰을 켜면 이미 와있습니다

1

스킬 갤러리에서 3개 설치

날씨 브리핑, 매크로 경제 리포트, RSS 뉴스 요약. 갤러리에서 클릭 한 번이면 설치 완료.

Terminal
$ kittypaw install weather-briefing
✓ 날씨 브리핑 설치 완료
$ kittypaw install macro-economy-report
✓ 매크로 경제 리포트 설치 완료
$ kittypaw install rss-digest
✓ RSS 뉴스 요약 설치 완료
2

텔레그램 토큰만 입력

각 스킬의 설정에 텔레그램 봇 토큰과 채팅 ID를 넣으면 끝. API 키는 날씨 스킬엔 필요 없습니다 (Open-Meteo 무료).

Terminal
$ kittypaw config
? 텔레그램 봇 토큰: ****
? 채팅 ID: ****
? 위치 (위도, 경도): 37.56, 126.97
✓ 설정 완료! 매일 07:00에 실행됩니다.
